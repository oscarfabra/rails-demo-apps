ASSUMPTIONS:
* File names and comments follow strictly the conventions used on the tutorial
  found at: http://rubytutorial.wikidot.com
* You have already installed ruby v1.8 or greater on your computer. For details
  on how to download and install Ruby please visit: http://ruby-lang.org/en/
* Remember that Ruby is an interpreted scripting language that doesn't need to
  be explicitly compiled before execution.

HOW TO EXECUTE:
* Just type the following on a command prompt or shell and hit enter:
# ruby filename.rb
